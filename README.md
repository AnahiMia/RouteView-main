# RouteView

Sitio web que permite consultar rutas de bus, ubicación en tiempo real,
horarios y guardar rutas favoritas.

## Autor
Mía Cayax - David Sanchez
Programación en ambiente web I

## Tecnologías
- HTML
- CSS
- JavaScript

## Funcionalidades
- Ubicación en tiempo real de buses
- Búsqueda y filtros de rutas
- Guardar rutas favoritas
- Alertas de estado del servicio
- Horarios actualizados
- Diseño responsive (celular, tablet, computadora)
